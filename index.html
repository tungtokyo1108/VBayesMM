Mathematical Foundations

VBayesMM builds upon rigorous Bayesian inference principles, incorporating spike-and-slab priors for sparse feature selection and variational inference for computational tractability in high-dimensional microbiome multiomics data.

Spike-and-Slab Prior Distribution

VBayesMM implements a sophisticated spike-and-slab prior for automatic feature selection, enabling probabilistic identification of keystone microbial species while maintaining sparsity in high-dimensional data.

Spike-and-Slab Prior for Taxonomic Units
$u_{il} | \gamma_{il} \sim \gamma_{il} \times \mathcal{N}(0, \sigma^2) + (1 - \gamma_{il}) \times \delta_0$ $\gamma_{il} \sim \text{Bernoulli}(\lambda), \quad l = 1, \ldots, L$

Where $\gamma_{il} \in \{0,1\}$ is a binary indicator for the $l$-th latent dimension of the $i$-th taxonomic unit, $\delta_0$ represents the Dirac spike at zero, and $\lambda$ is the prior inclusion probability hyperparameter.

Variational Bayesian Inference

To address computational intractability in high-dimensional settings, VBayesMM employs mean-field variational inference with carefully designed variational distributions for efficient posterior approximation.

Evidence Lower Bound (ELBO) Optimization
$$\mathcal{L}[q(\Xi|\Theta)] = \mathbb{E}_{q(\Xi|\Theta)}[\log p(\mathcal{D}, \Xi)] - \mathbb{E}_{q(\Xi|\Theta)}[\log q(\Xi|\Theta)]$$ $$\log p(\mathcal{D}) \geq \mathcal{L}[q(\Xi|\Theta)]$$
Variational Distributions
$q(U|\Theta) = \prod_{i=1}^{N} \mathcal{N}(u_i; \alpha_{U,i}, \text{diag}(\beta_{U,i}^2))$ $q(\gamma|\Theta) = \prod_{i=1}^{N} \prod_{l=1}^{L} \text{Bernoulli}(\gamma_{il}; \xi_{il})$ $q(V|\Theta) = \prod_{j=1}^{M} \mathcal{N}(v_j; \alpha_{V,j}, \text{diag}(\beta_{V,j}^2))$
Mean-Field Approximation KL Divergence Minimization Scalable Inference

Gumbel-Softmax Reparameterization

For discrete binary variables in the spike-and-slab prior, VBayesMM employs the Gumbel-Softmax trick to enable gradient-based optimization through continuous relaxation of discrete distributions.

Continuous Relaxation of Binary Indicators
$\tilde{\gamma}_{il} = \frac{\exp((\log \xi_{il} + \kappa_1)/\tau)}{\exp((\log \xi_{il} + \kappa_1)/\tau) + \exp((\log(1-\xi_{il}) + \kappa_2)/\tau)}$

Where $\kappa_1, \kappa_2 \sim \text{Gumbel}(0,1)$ are i.i.d. Gumbel random variables and $\tau$ is the temperature parameter controlling the approximation quality. As $\tau \to 0$, the relaxation approaches the discrete distribution.

Differentiable Discrete Sampling Temperature Annealing Gradient Estimation

Neural Network Architecture

VBayesMM extends the MMvec architecture with Bayesian neural networks, incorporating uncertainty quantification in both encoder and decoder networks for robust microbiome-metabolome relationship inference.

Conditional Probability Model
$p(Y_{kj} | i \sim \text{Categorical}(X_k)) = \frac{\exp(u_i^T v_j + v_{j0})}{\sum_{j'=1}^{M} \exp(u_i^T v_{j'} + v_{j'0})}$
Multinomial Likelihood
$p(Y_k | X_k) = \text{Multinomial}\left(w_k, \left(p(Y_{kj} | i \sim \text{Categorical}(X_k))\right)_{j=1}^{M}\right)$

VBayesMM Algorithm Workflow

1
Initialize Variational Parameters
Set initial values for $\Theta = \{\alpha_U, \beta_U, \xi, \alpha_V, \beta_V\}$ and hyperparameters
2
Sample from Variational Distributions
Use reparameterization trick: $u_i = \alpha_{U,i} + \beta_{U,i} \odot \epsilon$, where $\epsilon \sim \mathcal{N}(0,I)$
3
Apply Gumbel-Softmax for Feature Selection
Generate continuous relaxation $\tilde{\gamma}_{il}$ for binary indicators using Gumbel-Softmax with temperature $\tau$
4
Compute ELBO and Gradients
Evaluate $\mathcal{L}[q(\Xi|\Theta)]$ and compute $\nabla_\Theta \mathcal{L}$ using automatic differentiation
5
Update Variational Parameters
Apply Adam optimizer: $\Theta^{(t+1)} = \Theta^{(t)} + \eta \cdot \text{Adam}(\nabla_\Theta \mathcal{L})$
6
Convergence and Feature Extraction
Monitor ELBO convergence and extract posterior inclusion probabilities $\mathbb{E}[\gamma_{il}] = \xi_{il}$ for feature selection
